CURRENCY CONVERTER 

A modern, responsive Currency Converter web application built with vanilla HTML, CSS, and JavaScript. 
Designed with a clean UI, smooth animations, and real-time exchange rates.

✨ Features
🔄 Real-time Exchange Rates — Fetches live rates via ExchangeRate API
🌙 Dark Mode — Toggle between light and dark themes with smooth transitions
🧮 Built-in Calculator — Popup calculator for quick math before converting
🔁 Swap Currencies — Instantly swap From/To currencies with one click
🏳️ Flag Icons — Visual country flags for each currency
📱 Responsive Design — Works on all screen sizes
⚡ Smooth Animations — Polished transitions and hover effects throughout
